# ServiceNow Incident Management - Playwright Automation Framework

End-to-end automation framework for ServiceNow ITSM built with Playwright and TypeScript using Page Object Model (POM).

## What this covers
- Incident creation, assignment, escalation and closure workflows
- Dynamic element and iFrame handling
- 30+ automated test scenarios across 4 modules

## Tech Stack
Playwright · TypeScript · POM · GitHub Actions

## Project Structure
- /pages — Page Object classes
- /tests — Test files
- /data — Test data files
- /utility — Helper functions

## How to run
npm install
npx playwright test
